# Frontend (Next.js 14)

## Purpose
Serve the unified B2B+B2C dashboard, vendor managers, campaign views, analytics, and checkout experiences described in Phase 1.

## Getting Started
1. Install dependencies:
npm install
2. Run locally:
npm run dev
3. Point API calls to http://localhost:8000

## Focus Areas for Phase 1
- Build vendor/product lists highlighting ingestion status and deduplication events.
- Surface relationship data (lead stages, consent/opt-out) that the backend exposes.
- Provide B2B RFQ/cart flows and B2C checkout shells that can connect to mock APIs.
